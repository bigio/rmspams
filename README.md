<h1>rmspams</h1>
<p>Block spammers IP addresses from your junk folder.<br>
The only thing you need is Packet-Filter.</p>
</head>
<body>
<b>Installation</b>
<pre><code>Just run ./install.sh</code></pre>
<b>Usage</b>
<pre><code>rmspams [-i] [-d] email
<br>
-i : interactive mode
-d : enable debugging</code></pre>
